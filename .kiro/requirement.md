# Project Requirements: Expense & Budgeting Visualizer

## Core Functionalities
- **Real-time Dashboard**: Menampilkan total saldo (Total Balance) yang terupdate otomatis.
- **Transaction Management**: Fitur CRUD (Create, Read, Delete) untuk riwayat pengeluaran.
- **Geolocation Integration**: Deteksi lokasi otomatis (default: Jakarta) dan tampilan tanggal dinamis pada header.

## Optional Challenges (Completed)
1. **Appearance Toggle (Dark/Light Mode)**: 
   - Implementasi sistem tema menggunakan CSS Variables.
   - Fitur simpan preferensi tema ke `localStorage`.
2. **Flexible Taxonomy (Custom Categories)**: 
   - User dapat menambah dan menghapus kategori pengeluaran sendiri.
   - Kategori bersifat dinamis dan tersinkronisasi dengan dropdown input.
3. **Monthly Retrospective (Summary)**: 
   - Rekapitulasi otomatis total pengeluaran per bulan (Month & Year).
   - Tampilan ringkasan yang terpisah dari riwayat transaksi harian.
